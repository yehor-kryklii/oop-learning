# WHISH LIST

### Описание программы

```

Cоздать программу эмулирующую работу списка желаний.

```

### Описание методов

```

- Конструктор, принимающий: 
	- Имя хозяина списка желаний.

- Конструктор, принимающий:
	- Имя хозяина списка желаний.
	- Список инициализаторов состоящий из желаний.

- Оператор перемещающейго присваивания, а также конструктор перемещения.

- Запрещенный оператор копирующего присваивания, а также конструктор
	копий.

- Метод доступа к имени хозяина списка желаний.

- Метод позволяющий подсчитать количество желаний в списке.

- Метод добавления желания, принимающий:
	- Объект типа "Wish" (уже объявлен в заголовочном файле) 

- Метод добавления желание, принимающий:
	- Название желания.
	- Приоритетность желания.

- Метод проверки наличия желания, принимающий:
	- Название желания.

- Метод позволяющий получить приоритетность желания, принимающий:
	- Название желания.

- Метод позволяющий удалить желание из списка, принимающий:
	- Название желания.

- Метод позволяющий получить все желания с определенной приоритетностью:
	- Возвращает:
		+ Контейнер с желаниями (на выбор).
	- Принимает:
		+ Приоритетность желания.

- Метод позволяющий получить желания с наивысшей приоритетностью:
	- Возвращает:
		+ Контейнер с желаниями (на выбор).
	- Принимает:
		+ Количество необходимых желаний.

- Метод позволяющий получить желания с наименьшей приоритетностью:
	- Возвращает:
		+ Контейнер с желаниями (на выбор).
	- Принимает:
		+ Количество необходимых желаний.

- Метод позволяющий получить пересечение желаний двух списков:
	- Принимает:
		+ Другой список.
	- Возвращает:
		+ Множество строк с именами желаний.

- Метод позволяющий удалить желания с приоритетностью меньше 
	определенного значения.
	- Принимает значение приоритета.

- Перегруженный оператор += для добавления желания в список.


```
