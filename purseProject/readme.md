# PURSE

### Описание программы

```

Cоздать программу эмулирующую работу с дамской сумочкой (безграничный объем не учитывать)

```

### Описание методов

```

- Конструктор, принимающий: 
  - наименование сумочки
  - цвет (один из пяти доступных цветов) 
  - максимальное количество вещей в сумочке (эмулируем максимальный объем сумочки)
  - “грузоподъемность” сумочки (максимальный возможный вес вещей внутри сумочки) 
- Конструктор перемещения и перемещающий оператор присваивания.
- Удаленные конструктор копий и копирующий оператор присваивания. (Возможность создания точной копии сумочки вместе с вещами, как минимум, не верна)
- Метод позволяющий положить в сумочку новую вещь, принимающий:
  - наименование вещи
  - её вес
  - Метод позволяющий взять вещь из сумочки.
  - Метод получения веса вещи, принимающий имя предмета.
  - Метод доступа ко всем наименованиям вещей в сумочке (например печать на консоль)
  - Метод позволяющий проверить текущую загрузку и занятый объем.
  - Методы для доступа к имени сумочки, цвету и т.д.

```
